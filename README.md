# Dashboard Service Utilisateur — V1.18

Cette version apporte les évolutions suivantes :

- cartes RTU renommées `Service Utilisateur vVersion`, sans répétition du terme RTU ;
- dans les cartes RTU, Étape et Statut sont regroupés en haut à droite ;
- seule la date de réception reste affichée dans le corps de la carte, avec une barre de progression minimaliste `x/7` ;
- dans Réseau, Raccordement et Flux, le Statut est affiché sous l’Étape et le porteur est retiré de la carte ;
- le porteur est également retiré des cartes Chantiers de conception ;
- ajout d’un bouton `Réinitialiser les filtres` à côté de `Masquer les sujets communs`.

## Publication

Déposer `index.html`, `README.md` et `.nojekyll` à la racine du dépôt GitHub Pages, puis forcer le cache dans Grist avec un paramètre d’URL, par exemple `?v=1.18`.
